# AGENTS.md

## 固定要求

- 始终使用 MemOS MCP 检索与当前任务相关的记忆后再开始工作。
- 始终使用中文回答。
- 保持 Docker-first 的开发体验；不要假设本机 Python 已经加入 PATH。
- 仓库改动要包含 `.gitignore`，不要提交运行时数据库、缓存、反馈日志或构建产物。

## 项目约束

- 所有参数推荐必须附带相似案例依据。
- 所有实验反馈只能追加写入，不能覆盖历史记录。
- LLM provider 只能用于任务解析、工具编排和解释生成，不能直接凭空生成数值工艺参数。
- provider 切换必须通过配置完成，不要把供应商名称写死到业务逻辑。
- 原始 Excel 数据放在 `data/raw/`，清洗和建模产物放在 `data/interim/` 或 `data/processed/`。
